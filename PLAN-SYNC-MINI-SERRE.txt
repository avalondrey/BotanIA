PLAN D'IMPLÉMENTATION - Synchronisation Photo → Mini Serre
===========================================================

## PROBLÈME IDENTIFIÉ

Pour synchroniser intelligemment une photo de mini serre vers l'application :
1. Il faut savoir quel TYPE de plante correspond à chaque rang tracé
2. Les labels ("Poivrons", "Tomates") ne suffisent pas - il faut mapper vers plantDefId

## SOLUTION PROPOSÉE

### Phase 1 : Sélection du type de plante dans SeedRowPainter ✅ À IMPLÉMENTER

Quand l'utilisateur trace un rang et nomme "Poivrons" :
1. Modal affiche un **sélecteur de type de plante**
2. Liste déroulante avec toutes les plantes disponibles (tomato, pepper, lettuce...)
3. Stocke `plantDefId` dans l'objet SeedRow

**Modifications SeedRowPainter.tsx** :
```typescript
export interface SeedRow {
  id: string;
  color: string;
  points: { x: number; y: number }[];
  label?: string;
  plantDefId?: string;      // NOUVEAU
  plantCount?: number;
  createdAt: number;
}
```

Modal mise à jour :
- Dropdown "Type de plante" (pepper, tomato, lettuce, carrot, basil, strawberry)
- Champ "Nombre de plants" (existant)
- Champ "Nom personnalisé" (existant)

### Phase 2 : Fonction de synchronisation dans game-store.ts ✅ À IMPLÉMENTER

```typescript
importPlantsFromPhoto: (
  seedRows: SeedRow[], 
  miniSerreId?: string
) => {
  success: boolean;
  message: string;
  stats: { planted: number; skipped: number; errors: string[] };
}
```

**Logique** :
1. Vérifier qu'il y a une mini serre disponible
2. Pour chaque rang NON vide (color !== 'EMPTY') :
   - Vérifier que plantDefId existe
   - Vérifier qu'il y a assez de graines dans l'inventaire
   - Placer le nombre de plants demandé (plantCount)
3. Pour les croix (color === 'EMPTY') :
   - Laisser la case vide correspondante
4. Retourner statistiques : X plants ajoutés, Y cases vides

### Phase 3 : Mapping intelligent photo → grille 6×4

**Algorithme simple** :
- Grille mini serre = 6 rangées × 4 colonnes = 24 emplacements
- Tri des rangs par position Y (du haut vers le bas de la photo)
- Remplir ligne par ligne :
  - Rang 1 → Ligne 0 de la mini serre
  - Rang 2 → Ligne 1
  - etc.

**Algorithme avancé (Phase 3+)** :
- Analyser les coordonnées X,Y des traits
- Détecter rangs horizontaux vs verticaux
- Mapper proportionnellement vers la grille
