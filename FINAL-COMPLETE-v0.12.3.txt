## 🏷️ GrainTag (Photo → Jardin virtuel)

1. **Photo jardin**
   - Prends photo de tes rangs
   - GPS automatique

2. **Trace rangs**
   - Choisis couleur (🔴🔵🟢🟡...)
   - Trace avec doigt sur chaque rang
   - Marque les vides avec ❌

3. **Configure et synchronise**
   - Clique "🌱 Sync Jardin"
   - Choisis plante par rang (🍅 Tomate, 🫑 Poivron...)
   - Indique nombre de plants
   - Clique "Importer dans le Jardin"

4. **Résultat**
   - Plants placés en colonnes dans jardin virtuel
   - Chaque rang = une colonne
   - Espacement automatique
   - Visible dans Vue Plan


## ✨ Jumeau Numérique (Scan → Plant virtuel)

1. **Photo plante**
   - Prends photo d'UNE plante réelle
   - IA analyse : nom, stade, âge, santé

2. **Créer jumeau**
   - Bouton "✨ Créer Jumeau" apparaît
   - Clique dessus
   - Plant virtuel créé avec état réel

3. **Résultat**
   - Plant dans jardin avec bon stade/âge
   - Récompenses : +50🪙 +100⭐
   - Visible dans Vue Plan

## Jumeau Numérique - Données IA

```typescript
{
  plantName: "Poivron",
  confidence: 0.92,
  growthStage: {
    stage: 3,           // 0-5
    estimatedAge: 25    // jours
  },
  healthStatus: {
    isHealthy: true,
    diseaseName: "Sain"
  }
}
```

Récompenses :
- Base : +50🪙 +100⭐
- Si confidence > 0.8 : +25🪙 +50⭐
- Si stage ≥ 4 : +3 graines
