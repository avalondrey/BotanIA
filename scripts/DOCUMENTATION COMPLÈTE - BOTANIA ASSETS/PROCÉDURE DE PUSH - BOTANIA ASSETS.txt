## 🎨 Styles Graphiques

### 1. Paquets de Graines (Seed Packets)
**Style** : Manga cel-shade isométrique  
**Format** : 512x512px  
**Éléments** :
- Logo fournisseur en haut
- Nom de la variété
- Illustration du légume au centre
- Bordures noires épaisses
- Fond beige kraft

### 2. Cards Graines (Seed Cards)
**Style** : Manga cel-shade frontal  
**Format** : 512x512px  
**Éléments** :
- Illustration manga du légume
- Nom de la variété
- Badge fournisseur
- Style carte à collectionner

### 3. Plants - Stages de Croissance
**Style** : Manga cel-shade  
**Format** : 512x512px  
**Éléments** :
- Plante en pot terracotta
- 6 stages (jardin) ou 5 stages (mini-serre)
- Progression visible

### 4. Cards Shops
**Style** : Manga kawaii avec mascotte  
**Format** : 512x512px  
**Éléments** :
- Nom du shop coloré
- Mascotte personnage kawaii
- Bordures végétales décoratives

### 5. Équipements (Serres)
**Style** : Ornemental baroque vintage  
**Format** : Rectangulaire  
**Éléments** :
- Cadre ornemental complexe
- Titre en haut
- Vue perspective

## 📝 Prompts Templates

### Template Paquet de Graines

manga-style seed packet illustration, {SUPPLIER} brand logo at top,
"{VARIETY}" variety text, drawing of a {VEGETABLE} on the front of the packet,
cel-shaded, thick black manga borders, beige kraft paper background,
isometric angle, kawaii aesthetic, 512x512px, BotanIA game style



**Variables** :
- `{SUPPLIER}` : CLAUSE, KOKOPELLI, LE BIAU, LA FERME DE SAINTE MARTHE
- `{VARIETY}` : Nom de la variété
- `{VEGETABLE}` : Description du légume

### Template Card Graine

manga-style collectible card illustration, {VEGETABLE} {VARIETY},
{SUPPLIER} brand badge, cel-shaded, thick black manga borders,
beige kraft paper background, kawaii aesthetic,
512x512px, BotanIA game style






### Template Plant Stage

manga-style plant growth stage {STAGE_NUMBER} of {TOTAL_STAGES},
{VEGETABLE} {VARIETY} in terracotta pot, cel-shaded,
thick black manga borders, beige background,
kawaii aesthetic, 512x512px, BotanIA game style





### Template Card Shop

manga-style kawaii shop card, {SHOP_NAME} logo with colorful typography,
cute character mascot holding {PLANTS}, decorative floral border,
cel-shaded, beige kraft background, botanical illustrations,
512x512px, BotanIA game style




### Template Équipement (Serre)
vintage botanical encyclopedia card style, ornate baroque frame
with floral decorations, {EQUIPMENT_NAME} title at top,
greenhouse with multiple plants inside, perspective view,
open door, sunlight rays, detailed illustration,
beige aged paper background, 512x512px, BotanIA game style

## 🔄 Procédure de Génération

1. **Tester** un premier asset avec le prompt template
2. **Valider** le style avec l'équipe
3. **Générer** par lots (4-6 images)
4. **Vérifier** la cohérence
5. **Organiser** dans les dossiers
6. **Commit & Push** sur GitHub

Étape 4 : Commands Git

# 2. Ajouter tous les nouveaux fichiers
git add public/packets/
git add public/cards/
git add public/trees/
git add public/equipment/
git add docs/ASSETS_GENERATION.md

# 3. Vérifier ce qui va être commité
git status

# 4. Faire le commit
git commit -m "feat: Add seed packets, cards, and equipment assets

- Add 20 seed packets (Clause, Kokopelli, Le Biau, Sainte Marthe)
- Add shop cards for all suppliers
- Add tree cards (apple, lemon, cherry)
- Add mini-greenhouse equipment
- Add assets generation documentation

Style: manga cel-shade, 512x512px, kawaii aesthetic"

# 5. Push vers GitHub
git push origin main



# Si le fichier est trop gros, installez Git LFS
git lfs install
git lfs track "*.png"
git add .gitattributes
git add public/
git commit -m "feat: Add assets with Git LFS"
git push origin main
